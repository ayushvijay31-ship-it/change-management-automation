# Change Management Automation

AI-driven change management automation platform leveraging Claude for workflow decisions, approvals, and documentation.

---

## 🚀 Features

- Automated change request creation and tracking  
- AI-assisted approval recommendations  
- Risk assessment using AI  
- Workflow automation for approvals  
- Audit logs and reporting  

---

## 🛠 Tech Stack

- Python (Backend)
- FastAPI (API framework)
- Claude API (AI decision-making)
- GitHub (Version control)
